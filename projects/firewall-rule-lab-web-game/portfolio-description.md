# Portfolio Description

## Firewall Rule Lab: Interactive Cybersecurity Training Game

Firewall Rule Lab is an interactive web-based cybersecurity game that teaches firewall rule configuration through hands-on scenarios. Players review business requirements, build firewall rules, test simulated packets, tune detection settings, and explain why a firewall rule should be changed.

The game helps learners understand how firewall rules support daily operations while maintaining security. It includes scenarios for approved training video, email attachments, secure transactions, specific website access, guest Wi-Fi isolation, VPN access, cloud web applications, protected databases, and application firewall awareness.

This project demonstrates practical knowledge of firewall design, least privilege, access control lists, implicit deny, top-to-bottom rule evaluation, stateful and stateless firewall concepts, blocked services and websites, cloud firewall defaults, logging, alerting, and PCAP-style traffic review. It also demonstrates front-end development skills through a responsive browser-based interface, scoring system, rule builder, packet simulator, and reflection prompts.

## Short Portfolio Card Version

**Firewall Rule Lab** is a browser-based cybersecurity learning game that teaches users how to configure and test firewall rules. Players create allow/block rules, tune logging and alerts, and test simulated traffic against real business needs such as video access, email, secure transactions, guest Wi-Fi isolation, VPN access, and cloud application protection.

- Built an interactive rule builder for source, destination, protocol, port, action, and detection settings.
- Created easy-to-expert scenarios covering ACLs, implicit deny, least privilege, rule order, cloud firewalls, and PCAP-style review.
- Designed scoring and feedback so users can understand why specific firewall rules need to be reconfigured.
