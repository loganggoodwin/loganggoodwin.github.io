Portfolio Description

What this project is

Firewall Rule Lab is a web-based cybersecurity learning game that turns firewall configuration concepts into interactive practice. Players review a business scenario, build firewall rules, test simulated packets, evaluate PCAP-style evidence, and tune logging or alerting until the policy supports operations and security.

Why it helps

This project helps learners connect firewall rules to real operational needs, such as streaming approved training video, sending email attachments, completing secure transactions, accessing specific websites, isolating guest Wi-Fi, and protecting cloud resources. Instead of only reading about allow/block rules, users make decisions, see consequences, and explain why a rule should be changed.

Skills demonstrated

The game demonstrates rule design, least privilege, access control lists, top-to-bottom rule evaluation, implicit deny, stateful and stateless firewall concepts, detection tuning, logging, alerting, cloud firewall design, application firewall awareness, and basic front-end development.
