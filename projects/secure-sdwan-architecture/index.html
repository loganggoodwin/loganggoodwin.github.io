Overview

This project designs a future-state secure network architecture for a rapidly growing multi-site enterprise. The goal is to replace a fragile WAN with a scalable, fault-tolerant, security-first SD-WAN architecture that supports expansion, cloud integration, and collaboration workloads.

Business & Security Challenges

  • Inconsistent security controls across sites
  • Single points of failure in WAN connectivity
  • Lack of firewall protection at some locations
  • Limited scalability for rapid growth
  • Higher reliance on voice, video, and cloud services

Architecture Goals

  • High availability through redundancy and failover
  • Consistent security controls across all sites
  • Scalability for rapid user and site growth
  • Performance optimization for voice/video
  • Centralized visibility and monitoring
  • Zero Trust access principles

Proposed Network Architecture

Dual-Hub SD-WAN Topology

  • Primary hub: Dallas
  • Secondary / DR hub: Houston
  • Spoke sites: Memphis, Kansas City
  • Dual business-class ISP at hubs for failover and path selection

Security Enhancements

  • NGFW at every site
  • Zero Trust-aligned access model (ZTNA principles)
  • MFA for users and administrators
  • Central logging + monitoring (SIEM + NetFlow)
  • DPI and intrusion prevention

Segmentation & Access Control

  • Standard VLANs: Data, Voice, Video, Servers, Management, Guest
  • Inter-VLAN restrictions via ACLs and 802.1X/NAC where appropriate
  • Least privilege enforcement to reduce attack surface

Performance & Reliability

  • QoS for latency-sensitive collaboration traffic
  • Session Border Controller for VoIP security and stability
  • Redundant switching + multi-ISP connectivity

Outcome & Impact

  • Eliminates single points of failure
  • Standardizes security controls enterprise-wide
  • Improves collaboration performance
  • Supports growth while reducing operational/security risk

What I Would Improve Next

  • SIEM threat detection tuning and rules
  • Endpoint telemetry for stronger Zero Trust enforcement
  • Failover/DR simulations
  • Automated configuration compliance checks
