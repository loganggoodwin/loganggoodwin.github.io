# Portfolio Description

**Cyber Threat Sentinel** is an interactive web-based cybersecurity game that teaches real-time threat detection and incident response decision-making. Players review simulated alerts involving phishing emails, suspicious network traffic, endpoint compromise, brute-force login attempts, identity anomalies, ransomware behavior, and possible data exfiltration.

The project helps users learn because it turns cybersecurity concepts into hands-on decisions. Instead of only reading about phishing, malware, network scanning, or suspicious authentication, players must inspect the evidence and choose the proper response: allow safe activity, quarantine an email, block malicious traffic, isolate a compromised host, or escalate a serious incident.

This project fits well in a cybersecurity portfolio because it demonstrates both technical and instructional value. It shows front-end development skills, security awareness training design, incident response logic, user-centered interaction, and the ability to make cybersecurity concepts easier to understand through an engaging browser-based experience.
