# Packet Capture CTF Game

**Author:** Logan Garth Goodwin

Packet Capture CTF Game is a browser-based cybersecurity learning game that simulates basic packet analysis and capture-the-flag investigation. The player reviews simulated network packets, identifies suspicious traffic, and captures one flag per level.

## Project Profile Description

Packet Capture CTF Game is an interactive cybersecurity training project designed to help beginners practice packet-analysis thinking in a safe browser environment. The game includes five levels focused on common network-security scenarios: suspicious DNS activity, HTTP data exfiltration, TCP port scanning, ICMP probing, and SSH brute-force behavior. Players choose Easy, Medium, or Hard mode, review packet details such as protocol, source IP, destination IP, port, and packet summary, then capture the packet that best matches the mission clue.

This project demonstrates front-end web development, cybersecurity awareness, user interface design, and gamified technical training. I built it as a simple online learning tool that connects networking concepts with hands-on decision-making.

## Features

- Five packet-capture levels
- Five CTF flags
- Easy, Medium, and Hard difficulty modes
- Simulated packet cards with protocol, IP address, port, and packet information
- Score, timer, error counter, and captured-flag tracking
- Analyst console log
- Works in a browser with no installation required

## How to Play

1. Open `index.html` in a web browser.
2. Choose Easy, Medium, or Hard.
3. Click **Start Game**.
4. Read the mission clue.
5. Review each packet card.
6. Click the packet that best matches the suspicious activity.
7. Capture all five flags to complete the game.

## How to Put It Online with GitHub Pages

1. Create a new GitHub repository.
2. Upload `index.html` and this `README.md` file.
3. Go to **Settings** in the repository.
4. Click **Pages**.
5. Under **Build and deployment**, choose **Deploy from a branch**.
6. Choose the `main` branch and `/root` folder.
7. Click **Save**.
8. GitHub will give you a public website link after it publishes.
