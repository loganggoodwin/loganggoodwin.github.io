Packet Capture CTF Game is an interactive cybersecurity training project designed to help beginners practice packet-analysis thinking in a safe browser environment. The game includes five levels focused on common network-security scenarios: suspicious DNS activity, HTTP data exfiltration, TCP port scanning, ICMP probing, and SSH brute-force behavior. Players choose Easy, Medium, or Hard mode, review packet details such as protocol, source IP, destination IP, port, and packet summary, then capture the packet that best matches the mission clue.

This project demonstrates front-end web development, cybersecurity awareness, user interface design, and gamified technical training. I built it as a simple online learning tool that connects networking concepts with hands-on decision-making.
