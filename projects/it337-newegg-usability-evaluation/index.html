Overview

This project evaluates the Newegg website using a structured usability test plan plus a heuristic review. I believe strong UX reduces user error and improves system effectiveness.

What This Covers

  • Test objectives and tasks: search, compare, and navigate to checkout
  • Heuristic review aligned to usability best practices
  • User test results with completion outcomes and satisfaction ratings
  • Recommendations to improve discoverability and reduce friction

Skills Demonstrated

  • User-centered evaluation and requirements thinking
  • Structured findings and actionable recommendations
  • Documentation and stakeholder-friendly communication

What I Would Improve Next

  • Add severity ratings for each finding
  • Include before/after screenshots per recommendation
  • Expand accessibility checks (WCAG-style)
