# SQL Database Troubleshooter: Ops Lab

A single-page web game that teaches SQL database setup, troubleshooting, permissions, performance tuning, and maintenance.

## How to Play

1. Open `index.html` in a web browser.
2. Pick a difficulty:
   - **Easy:** shows the exact command.
   - **Normal:** gives partial hints.
   - **Hard:** gives the scenario only, so you must know the commands.
3. Type commands into the simulated SQL console.
4. Keep the Database Care meter high by using the correct commands.

## Missions

- Build the Client Portal Database
- Secure the Application User
- Fix a Database Connection Outage
- Repair a Permission Denied Error
- Tune a Slow Customer Order Query
- Run a Backup and Integrity Check

## Notes

This is a browser-based training simulation. It does not connect to a real database and is safe to run locally.
