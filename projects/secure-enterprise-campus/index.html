Secure Enterprise Campus Infrastructure

Cisco Packet Tracer · Cisco IOS · ASA Firewall · VLAN Segmentation · ACLs

ProjectsHome

This project demonstrates the design and deployment of a hardened, multi-segment campus network for a mid-sized organization with roughly 70 endpoints. The architecture uses a hierarchical collapsed core model to balance Layer 3 performance with a defense-in-depth posture.

Executive summary

Key outcomes included isolation of guest traffic, creation of a DMZ for public-facing services, and use of Zero Trust principles at the access layer. I think this project works well on a portfolio because it shows both architecture planning and control-level implementation.

Security controls

Layer 2 hardening

  • Port security on all access switches to mitigate unauthorized hardware insertion and MAC spoofing.
  • DHCP snooping to help prevent rogue DHCP and man-in-the-middle scenarios.
  • Native VLAN moved away from VLAN 1 to reduce VLAN hopping risk.
  • BPDU Guard and PortFast on edge ports for cleaner access-layer behavior.
interface range fa0/1-24
 switchport mode access
 switchport access vlan 10
 switchport port-security
 switchport port-security maximum 1
 switchport port-security violation shutdown
 switchport port-security mac-address sticky
 spanning-tree portfast
 spanning-tree bpduguard enable

Layer 3 least privilege

Extended ACLs enforce guest isolation and management-plane restrictions. Guest users are blocked from internal RFC1918 access while still allowed DNS and web traffic outward.

ip access-list extended GUEST_ISO_ACL
 permit udp 10.10.40.0 0.0.0.255 any eq 53
 permit udp any any eq 67
 deny ip 10.10.40.0 0.0.0.255 10.10.0.0 0.0.255.255
 permit ip any any

interface vlan 40
 ip access-group GUEST_ISO_ACL in

Perimeter and DMZ controls

  • Cisco ASA firewall providing stateful packet inspection.
  • Inbound traffic limited to TCP 80 and 443 for public-facing services.
  • DMZ prevented from initiating new connections to inside networks.

Verification

  • Rogue device testing triggered err-disable on protected ports.
  • Pings from Guest to Server VLAN were dropped while outbound internet remained available.
  • Stateful return traffic from the internet was allowed while unsolicited inbound traffic was dropped.

  • ACL authoring, ASA firewall rules, and DMZ design.
