# Fayetteville Office Network Design Lab

Interactive browser-based infrastructure planning lab based on the Network Infrastructure Proposal: Fayetteville Office project.

## What it teaches

- Star topology design
- Business ISP and uptime planning
- Managed switching and cabling supportability
- Firewall placement and practical segmentation
- BitLocker, VPN, and endpoint security baselines
- Network printing and teleconferencing planning
- Scalability, documentation, and support handoff

## How to run

Open `index.html` in a browser. No installation is required.
