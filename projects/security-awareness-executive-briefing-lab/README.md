# Security Awareness Executive Briefing Lab

**Author:** Logan Garth Goodwin

Security Awareness Executive Briefing Lab is an interactive browser-based cybersecurity training simulator built as a companion to the Security Awareness Executive Summary portfolio project.

## Purpose

This lab turns a leadership security awareness presentation into an interactive decision simulator. Users play the role of an IT/security advisor and make practical decisions involving phishing, MFA, sensitive data, social engineering, insider risk, security metrics, and executive sponsorship.

## What it teaches

- Human risk and security awareness
- Policy communication
- Executive decision-making
- Phishing and social engineering response
- Least privilege and access review
- Security culture and reporting behavior
- Risk-based program metrics

## Features

- Seven realistic workplace security scenarios
- Risk, executive readiness, security culture, and budget meters
- Instant feedback after each decision
- Final executive-style report
- Print/save report option
- Fully browser-based design with no installation required

## How to run

Open `index.html` in a browser.

For GitHub Pages, place this folder in a repository or inside a portfolio folder and link to:

```text
projects/security-awareness-executive-briefing-lab/index.html
```
