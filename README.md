# Logan G. Goodwin Portfolio Site

A polished GitHub Pages portfolio focused on cybersecurity, infrastructure, networking, technical documentation, and active CTF work.

## What is included

- Updated homepage with featured work, active projects, certifications, experience, and a portfolio status section
- Projects landing page with duplicate Information Assurance card removed
- CTF hash-cracking write-up added as current work
- Open Graph preview image added for better LinkedIn and social sharing

## Deploy on GitHub Pages

1. Copy the contents of this folder into the root of your `loganggoodwin.github.io` repository
2. Commit and push to GitHub
3. GitHub Pages will publish the site from the repository root

## Final manual steps

### 1. Configure the contact form
Open `script-updated.js` and replace `YOUR_FORM_ID` with your Formspree ID. Until that is done, the form uses a `mailto:` fallback.

### 2. Replace illustrative CTF examples later
The CTF write-up currently uses illustrative hash values and sample cracked output. Once the Cyber Skyline competition is complete and results are public, replace those examples with the real hashes and real cracked output.
